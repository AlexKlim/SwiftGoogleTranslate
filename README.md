A framework to use <a href="https://cloud.google.com/translate/docs/reference/rest">Cloud Translation API by Google</a>.

## Installation
<b>CocoaPods:</b>
<pre>
pod 'SwiftGoogleTranslate'
</pre>
<b>Manual:</b>
<pre>
Copy <i>SwiftGoogleTranslate.swift</i> to your project.
</pre>
